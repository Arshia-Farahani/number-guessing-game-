# number-guessing-game-
There are two players in this game.
  Player one must choose a number from 1 to 100.
Then player two must guess the number chosen by player one.
If the number guessed by player two is smaller than the number of player one, then player two is said to have the number guessed smaller, and if the number guessed by player two is greater than the number of player one, then player two is said to have the number guessed. is bigger
Player two has a limited number of chances.
